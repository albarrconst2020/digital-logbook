# Digital Logbook

This is a **Digital Logbook** web application built with [Next.js](https://nextjs.org) for tracking vehicle entries, exits, and registration status.  
It was bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

---

## Features

- Log vehicle entries and exits
- Track registered vs unregistered vehicles
- Display vehicle duration inside the premises
- Search and filter vehicles by duration and status
- Visual charts for traffic analytics
- Responsive and mobile-friendly interface

---

## Technologies Used

- **Frontend:** Next.js, React, Tailwind CSS
- **Backend / Database:** Firebase Firestore
- **Authentication:** Firebase Authentication
- **Charts & Visualization:** Chart.js
